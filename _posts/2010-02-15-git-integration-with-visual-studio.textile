I started to use <a href="http://git-scm.com/" target="_blank">Git</a> seriously 5 months ago when I sign up for my <a href="http://github.com/" target="_blank">Github</a> account on September 1st 2009. Before that I was a <strike>very</strike> happy subversion user. One of the things that I really like about Git is the speed. 
 I decided to use Git from the command line to learn and understand the commands really well. I occasionally used the GUI to perform some tasks that seemed easier to do that way.
 <h5>Git within VS.</h5> Even when I really enjoy working from the command line I know that some developers prefer to stay within the IDE and I understand that. Yesterday while installing VS 2010 RC in my newly paved laptop I decided to check out the <a href="http://sourceforge.net/projects/gitextensions/" target="_blank">Git Extensions</a>. 
 After installing it VS presents a new menu item for Git.
