  Ok, the patch can be downloaded from the <a href="http://code.google.com/p/bigvisiblecruise/issues/detail?id=32" target="_blank">Issue tracker of the project</a>. This support the following notation in the app.config.
  <pre class="xml" name="code">&amp;lt;setting name=&amp;quot;IncludedProjects&amp;quot; serializeAs=&amp;quot;Xml&amp;quot;&amp;gt;<br />  &amp;lt;value&amp;gt;<br />    &amp;lt;ArrayOfString xmlns:xsi=&amp;quot;http://www.w3.org/2001/XMLSchema-instance&amp;quot;<br />      xmlns:xsd=&amp;quot;http://www.w3.org/2001/XMLSchema&amp;quot;&amp;gt;<br />    &amp;lt;string&amp;gt;CCNet&amp;lt;/string&amp;gt;<br />    &amp;lt;string&amp;gt;Ivy.NET&amp;lt;/string&amp;gt;<br />    &amp;lt;/ArrayOfString&amp;gt;<br />  &amp;lt;/value&amp;gt;<br />&amp;lt;/setting&amp;gt;</pre>

Where each &amp;lt;string&amp;gt; is the name of a project. The project names are case sensitive. If you set a RegEx to filter the projects, the expression will be evaluated first.


This patch have been done over the Revision 125 of the Trunk.


Enjoy!
