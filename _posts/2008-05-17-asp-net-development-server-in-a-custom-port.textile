During Scott's Hanselmann presentation on Dynamic Data at the TVBUG he show a little trick that I didn't know about and I want to show it here.It always annoy me that every time you click run ina a web application using VS it runs using the ASP.Net development server and starts in a random port.So if you want to make sure that the projects starts always at the same port, all you need to do is right click on the solution and select propertiesIn the property sheet for the solution click on the Web tab:Change the default Auto-assign Port to Specific port and type a port of your liking.<br><br><img src="images/posts/sp32-20080517-140354.png" alt="" border="0">
